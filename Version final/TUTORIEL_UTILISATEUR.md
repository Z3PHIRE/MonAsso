# Tutoriel MonAsso

Ce guide explique comment utiliser MonAsso rapidement, sans jargon technique.

## 1) Premiere ouverture
- Ouvrez MonAsso.
- Verifiez la barre du haut: le mode est local hors ligne.
- Si le menu gauche est masque, cliquez sur `Afficher menu`.

## 2) Ajouter des personnes
- Ouvrez `Personnes`.
- Renseignez les champs principaux: prenom, nom, type, telephone, email, actif.
- Utilisez `Informations avancees` uniquement si necessaire.
- Cliquez sur `Creer`.

## 3) Creer vos categories et sous-categories
- Ouvrez `Parametres`.
- Allez dans `Categories personnalisables`.
- Creez une categorie racine (ex: Materiel).
- Creez ensuite une sous-categorie avec le champ `Parent` (ex: Materiel > Sonorisation).
- Activez les cibles souhaitees: Personne, Evenement, Reunion, Tache.
- Revenez dans vos fiches (Personnes / Evenements) puis utilisez `Gerer categories` ou l'onglet categories.

## 4) Gerer les evenements
- Ouvrez `Evenements`.
- Saisissez titre, date, heure, lieu, responsable, statut.
- Utilisez la section avancee pour logistique, partenaires et notes.
- Dans `Suivi`: participants, budget, taches, checklist, documents, categories.

## 5) Gerer les reunions
- Ouvrez `Reunions`.
- Renseignez titre, date, horaires, lieu, organisateur, responsable.
- Ajoutez les participants et la checklist dans l'onglet `Suivi`.

## 6) Gerer les taches
- Ouvrez `Taches`.
- Creez une tache avec responsable, echeance, priorite, statut.
- Filtrez par responsable, statut et date limite.

## 7) Utilisation quotidienne
- Ouvrez `Utilisation quotidienne`.
- Regardez d'abord:
  - aujourd'hui
  - cette semaine
  - taches urgentes
  - prochains evenements et reunions
- Utilisez les actions rapides pour creer un evenement, une tache ou une personne.

## 8) Personnaliser le menu gauche
- Cliquez `Personnaliser le menu` dans la barre gauche.
- Cochez/decohez les modules a afficher.
- Utilisez `Reordonner` pour monter/descendre un module.
- Si la liste depasse, `Autres modules` regroupe les ecrans restants.

## 9) Sauvegardes et donnees demo
- Ouvrez `Parametres` > `Securite des donnees`.
- Creez une sauvegarde regulierement.
- Les donnees de demonstration sont disponibles sur base vide.
